# mew
mew is a simple menu for Wayland.

## Building
In order to build mew, ensure that you have the following dependencies:

* fcft
* pkg-config
* wayland
* wayland-protocols
* xkbcommon

Afterwards enter the following command to build and install mew
(if necessary as root):
```
make
make install
```
